DatabaseRepository

class DatabaseRepository(context: Context)
Content copied to clipboard

Repository for interacting with different databases. Combines access to the retrofit interface and the local room database. Stores only simple requests as Suspendfunctions with the withcontext-Scope,specific implementations are located in the ViewModels com.fachhochschulebib.fhb.pruefungsplaner.viewmodel.BaseViewModel. Required for the MVVM-Pattern.
