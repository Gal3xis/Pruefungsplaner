RecyclerViewFavoriteAdapter

class RecyclerViewFavoriteAdapter(context: Context, entryList: MutableList<TestPlanEntry>, viewModel: BaseViewModel) : RecyclerView.Adapter<RecyclerViewFavoriteAdapter.ViewHolder>
Content copied to clipboard

The RecyclerView.Adapter for the RecyclerView that holds information about the favorit exams. The information is stored in multiple List-Objects, each holding one kind of information for every exam that needs to be displayed. E.g. the exam at position 1 gets his information from every list at index 1.
